Problem 2904. -- 平方和

2904: 平方和

Time Limit: 10 Sec  Memory Limit: 128 MB
Submit: 174  Solved: 78
[Submit][Status][Discuss]

Description

给定一个整数N，求N最少可以拆成多少个完全平方数的和。 
 

Input


第一行一个整数TEST，表示数据组数。
接下来TEST行，每行一个整数N。 
 

Output

TEST行，每行一个正整数，表示N最少可以拆成多少个完全平方数的和。
数据规模
30.的数据N≤10^6 
60.的数据N≤10^9
100.的数据0≤N≤10^18 ,TEST<=1000
