Problem 4800. -- [Ceoi2015]Ice Hockey World Championship

4800: [Ceoi2015]Ice Hockey World Championship

Time Limit: 10 Sec  Memory Limit: 256 MB
Submit: 570  Solved: 295
[Submit][Status][Discuss]

Description

有n个物品，m块钱，给定每个物品的价格，求买物品的方案数。

Input

第一行两个数n，m代表物品数量及钱数
第二行n个数，代表每个物品的价格
n<=40，m<=10^18

Output

一行一个数表示购买的方案数
（想怎么买就怎么买，当然不买也算一种）

Sample Input

5 1000
100 1500 500 500 1000

Sample Output

8
