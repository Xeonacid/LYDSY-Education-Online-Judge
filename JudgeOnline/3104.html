Problem 3104. -- bracket

3104: bracket

Time Limit: 10 Sec  Memory Limit: 128 MB
Submit: 17  Solved: 3
[Submit][Status][Discuss]

Description

给出一个表达式，只有加减乘除运算，还有一个未知数
x.现要求将其中不必要的括号全部去掉，而且与先前的
那个表达式等价。

Input

一个数字N，代表有多少组数据。

下面N行每行一个长度不超过10^6的表达式

Output

N行，每行一个表达式

Sample Input

2
x+(x+(x+x)-(x*x))/x
(x*x)/((x*x))+(x)

Sample Output

x+(x+x+x-x*x)/x
x*x/(x*x)+x
