Problem 3917. -- [Baltic2014]sequence

3917: [Baltic2014]sequence

Time Limit: 10 Sec  Memory Limit: 256 MB
Submit: 321  Solved: 141
[Submit][Status][Discuss]

Description

 序列A由从N开始的连续K个数按顺序构成,现在将A中的每个数只保留某一个数码,记为序列B,给定K和B,求可能的最小的N

Input

第一行一个数K,第二行K个数B_i

Output

输出一个数N

Sample Input

6
7 8 9 5 1 2

Sample Output

47

HINT

K<=100000,0<=B_i<=9

N是正整数
