Problem 3915. -- [Baltic2014]demarcation

3915: [Baltic2014]demarcation

Time Limit: 30 Sec  Memory Limit: 256 MBSec  Special Judge
Submit: 3  Solved: 2
[Submit][Status][Discuss]

Description

有一个简单多边形,它的每个顶点坐标都是整数,而且每条边都是水平或竖直的,现在想用一条线段将它分为两个全等的多边形,线段必须是平行或竖直的.

Input

  第一行1个数n,表示顶点数
  接下来n行,每行一个点,按逆时针描述这个图形

Output

若答案不存在,输出"NO",否则输出X1,Y1,X2,Y2表示(X1,Y1)-(X2,Y2)这条线段

Sample Input

10
0 0
1 0
1 1
3 1
3 5
2 5
2 3
1 3
1 2
0 2

Sample Output

1 2 3 2

HINT

对于100%的数据 4 ≤ N ≤ 100000.
