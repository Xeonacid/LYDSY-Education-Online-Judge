Problem 4544. -- 椭圆上的整点

4544: 椭圆上的整点

Time Limit: 30 Sec  Memory Limit: 128 MB
Submit: 239  Solved: 127
[Submit][Status][Discuss]

Description

求不定方程X^2+3*Y^2=N^2整数解的个数。

Input

第一行一个整数T ( T <= 10 )，表示输入包含T组数据。
接下来T行每行一个正整数n

Output

输出应包含T行，每行一个整数N(n <= 10^12)，表示该组数据的答案。

Sample Input

3
5
1001
191752422

Sample Output

2
18
162

HINT
