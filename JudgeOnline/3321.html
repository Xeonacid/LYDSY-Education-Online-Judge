3321: 生成树Stcnt

Time Limit: 10 Sec  Memory Limit: 512 MB
Submit: 118  Solved: 73
[Submit][Status][Discuss]

Description

  给定两个数 n; k ，求出满足条件的树的个数：

1.     有 nk 个点，标号为 0; 1,…… nk

2.     如果i≡j mod N,则I,j间不存在边

Input

 

一行两个数N,K

Output

一行一个数，表示答案Mod 10^9+7

 

Sample Input

2 2

Sample Output



4


HINT



N,K<=10^9
