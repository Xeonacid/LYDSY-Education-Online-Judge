Problem 5459. -- Sum

5459: Sum

Time Limit: 15 Sec  Memory Limit: 256 MB
Submit: 23  Solved: 3
[Submit][Status][Discuss]

Description

求有多少 N 位十进制数是 P 的倍数且每位之和小于等于 M，允许前导 0，答
案对 998244353 取模。

Input

一行三个整数 N,P,MM。 MM 的含义见输出格式。
1<=N<=10^9.1<=P<=16.1<=MM<=1000

Output

一行输出 MM+1 个整数，第 i 个(从 0 开始)表示 M=i 时的答案

Sample Input

2 3 3

Sample Output

1 1 1 5
