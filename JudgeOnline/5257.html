Problem 5257. -- yi

5257: yi

Time Limit: 40 Sec  Memory Limit: 256 MB
Submit: 19  Solved: 3
[Submit][Status][Discuss]

Description

给定一个正整数x，对它进行变换，如果它是奇数，那么变成3x+1，否则变成x/2
问经过多少次变换这个数字会变成1。 

Input

 一行一个长度为n的01串，表示x的二进制形式，没有前导0。

n<=300000,01串除了第一位均为随机生成

Output

一行一个正整数，表示答案。

Sample Input

1010

Sample Output

6
