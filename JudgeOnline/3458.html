Problem 3458. -- Luck

3458: Luck

Time Limit: 10 Sec  Memory Limit: 256 MB
Submit: 17  Solved: 11
[Submit][Status][Discuss]

Description

S国是一个爱好数字的国度，总有些人会夜以继日的数着数字以占卜凶吉，凡是数字在十进制表示之下有连续k个4或k个7相连则被认为是幸运的。他们看遍了1到n之间的所有数，想考一考你。因为他们预先知道答案，所以他们只需你告诉他们幸运的数字mod 109+7就行。

Input

 
　　第一行一个数k。
　　第二行一个数n。

Output

　　一个数，如题所述。

Sample Input

1
100

Sample Output

36

HINT



　　设t为n的位数

　　100%的数据1<=k<=t<=106
