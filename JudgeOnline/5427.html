Problem 5427. -- 最长上升子序列

5427: 最长上升子序列

Time Limit: 10 Sec  Memory Limit: 256 MB
Submit: 1  Solved: 1
[Submit][Status][Discuss]

Description

现在给你一个长度为n的整数序列，其中有一些数已经模糊不清了，现在请你任意确定这些整数的值，
使得最长上升子序列最长。（为何最长呢？因为hxy向来对自己的rp很有信心）

Input

第一行一个正整数n
接下来n行第i行格式如下
K x：表示第i个数可以辨认且这个数为x
N：表示第i个数一个已经辨认不清了
n<=100000,|x|<=10^9

Output

一个正整数代表最长上升子序列最长是多少

Sample Input

4
K 1
N
K 2
K 3

Sample Output

3
【样例说明】
当序列为1 1 2 3 （也可以1 2 2 3,1 0 2 3……）
时最长上升子序列最长，为3
