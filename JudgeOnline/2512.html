Problem 2512. -- Groc

2512: Groc

Time Limit: 2 Sec  Memory Limit: 128 MB
Submit: 5  Solved: 4
[Submit][Status][Discuss]

Description

Input

第一行有四个整数P,N,A，B
以下P行，每个两个整数描述一个物品，即这个物品的走道和从走到底端到这个物品的距离

Output

一个整数，即最小路程

Sample Input

7 5 10 3
2 8
3 3
3 5
3 7
4 10
5 10
4 3

Sample Output

54

HINT



1<=P<=300

1<=N<=350

1<=A<=25

1<=B<=5
