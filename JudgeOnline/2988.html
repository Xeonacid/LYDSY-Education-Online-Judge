Problem 2988. -- DIVISORS

2988: DIVISORS

Time Limit: 10 Sec  Memory Limit: 256 MB
Submit: 5  Solved: 1
[Submit][Status][Discuss]

Description

 

给定正整数B，X（B<=10^12,X<=60）,求有多少个N满足NX存在因子大于N且小等于B。

Input

多组数据，每个测试点第一行一个数T<=40表示数据的组数
    接下来T行每行两个数，分别是给定的两个数B,X

Output

    能使小z获胜的方案数
数据范围
    100% B<=10^12，X<=60

Sample Input

3
5 1
10 3
100 6




Sample Output

0
5
63
