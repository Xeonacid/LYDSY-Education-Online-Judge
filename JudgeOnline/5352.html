Problem 5352. -- Number Theory

5352: Number Theory

Time Limit: 10 Sec  Memory Limit: 512 MB
Submit: 6  Solved: 0
[Submit][Status][Discuss]

Description

N^2n+N^m≡x Mod P
对于给定的x，m，p，p 是质数，求一个可行的正整数n，满足上式
由于解不唯一，所以这题设有Special Judge，支持检验高精度的n

Input

 第一行三个非负整数x，m，p

x<=p
m<=P
P<=10^9+7,P为随机生成大质数

Output

输出任意一个合法的n（数据保证有解）

Sample Input

17 0 997

Sample Output

2

HINT

 请不要提交！

by AekdyCoin
