Description

 

Input

输入文件第一行包含两个空格分开的正整数n和m，分别表示方格的行数和列数。

接下来n行，每行m个字符，为"."或"#"。"#"表示该方格为装饰性图形，"."表示可以放置L图案的空白区域。

Output

输出一个整数，为可能的logo总数。

Sample Input

4 4
....
#...
....
..#.

Sample Output

4

HINT

 对于100%的数据，2≤n,m≤30
