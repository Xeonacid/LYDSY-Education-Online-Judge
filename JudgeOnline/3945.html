Problem 3945. -- 无聊的邮递员

3945: 无聊的邮递员

Time Limit: 20 Sec  Memory Limit: 512 MB
Submit: 53  Solved: 15
[Submit][Status][Discuss]

Description

Input

第一行两个整数n, k，意义如问题描述。
接下来n行，每行一个整数x_i。

Output

一行一个整数，表示第k小的方案的路径总长度。数据保证至少存在k个方案。

Sample Input

5 11
1
-1
2
-2
3

Sample Output

11

HINT

N<=10000,K<=500000,|X_i|<=100000000

Source

2014年国家集训队十五人互测
