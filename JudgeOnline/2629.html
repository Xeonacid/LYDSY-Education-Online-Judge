Problem 2629. -- binomial

2629: binomial

Time Limit: 30 Sec  Memory Limit: 128 MB
Submit: 104  Solved: 43
[Submit][Status][Discuss]

Description

对于给定的n和p，求对于所有的0<=i<p,满足C(n,k)%p=i的k的个数
　　注：C(n,k)=n!/(k!*(n-k)!)

Input

仅一行包含两个正整数n和p

Output

仅一行，为一个长度为p的字符串s，s[i]表示C(n,k)%p=i的k的个数除以29后的余数，s[i]视为一个29进制的数字

Sample Input

20 4

Sample Output

D440

HINT

n<p^10 p=51061
