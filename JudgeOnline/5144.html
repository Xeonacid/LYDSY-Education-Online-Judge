珂朵莉给了你一个长为n的序列a，有m次操作
1.把区间[l,r]内所有数都加上x
2.查询区间[l,r]内的最大子段和，可以不选数

Input

第一行两个数n,m
第二行n个数表示序列a
后面m行
1 l r x : 把区间[l,r]内所有数加上x
2 l r : 查询区间[l,r]的最大子段和
1 <= n , m <= 100000 , | ai | <= 2000000000 , 0 <= x <= 1000000
共15组数据

Output

对于每个询问，输出一个数表示答案

Sample Input

5 5
-2 -3 -3 -3 -3
2 1 5
1 2 4 4
2 1 5
1 2 3 1
2 3 3

0
3
2
