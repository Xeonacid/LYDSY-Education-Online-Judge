Problem 3100. -- 排列

3100: 排列

Time Limit: 10 Sec  Memory Limit: 16 MB
Submit: 765  Solved: 214
[Submit][Status][Discuss]

Description

给定一个长度为n的序列a，选取连续的一段使其为1~k的一个排列。
求k的最大值。

Input

输入的第一行包含一个整数n。接下来n个数描述序列a
1<=N<=1000000，1 <= ai <=n

Output

输出一个整数表示k的最大值。

Sample Input

5
1 2 3 4 5

Sample Output

5
