Problem 1320. -- Pku3530A Modular Arithmetic Challenge

1320: Pku3530A Modular Arithmetic Challenge

Time Limit: 1 Sec  Memory Limit: 64 MB
Submit: 139  Solved: 47
[Submit][Status][Discuss]

Description

求3<=(3*x) Mod 5<=4的最小正整数解

Input

第一行一个整数。T代表数据组数。 接下来T行，每行4个数，M，D，L，R表示L<=(D*x Mod M)<=R。

Output

共T行，表示x的最小正整数解，若无解则输出-1。

Sample Input

1
5 3 3 4

Sample Output

1


HINT

T<=100
1<=M、D、L、R<=1000000000
