Problem 5088. -- HDU 6000 Wash

5088: HDU 6000 Wash

Time Limit: 10 Sec  Memory Limit: 512 MB
Submit: 107  Solved: 49
[Submit][Status][Discuss]

Description

你现在要洗L件衣服。你有n台洗衣机和m台烘干机。由于你的机器非常的小，因此你每次只能洗涤(烘干)一件衣服
。第i台洗衣机洗一件衣服需要wi分钟，第i台烘干机烘干一件衣服需要di分钟。请问把所有衣服洗干净并烘干，最
少需要多少时间？假设衣服在机器间转移不需要时间，并且洗完的衣服可以过一会再烘干

Input

输入文件的第一行有3个整数L,n和m。
第二行有n个整数w1,w2,...,wn。
第三行有m个整数d1,d2,...,dm。
L<=10^6,N<=10^5,M<=10^5

Output

输出一行一个整数，表示所需的最少时间。

Sample Input

1 1 1
1200
34

Sample Output

1234

HINT
