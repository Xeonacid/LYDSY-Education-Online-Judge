Problem 1371. -- [Baltic2003]Barrel

1371: [Baltic2003]Barrel

Time Limit: 5 Sec  Memory Limit: 64 MB
Submit: 41  Solved: 12
[Submit][Status][Discuss]

Description

给出一个水池的底面积及高度。 现在把一些立方体放到其中，问最终水平面的高度. 注意: 1:水的密度为1.0 2:空气的影响忽略不计 3:立方体完全放到水池中。 4:立方体不会旋转且相互不会接触 。

Input

第一行给出S,H,V，（0

Output

最终水平面的高度，答案误差不超过10^-4

Sample Input

100 10 500
1
1 0.5

Sample Output

5.0050
