Description

Input

第1行包括两个正整数N，K，表示了棋盘的列数和放的车数。
第2行包含N个正整数，表示了棋盘每列的高度。

Output

包括一个非负整数，表示有多少种放置的方案，输出答案mod
1000000007后的结果即可。 
 

Sample Input

2 3 1 2 4

Sample Output

43
