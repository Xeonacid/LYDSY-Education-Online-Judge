Problem 3329. -- Xorequ

3329: Xorequ

Time Limit: 1 Sec  Memory Limit: 256 MB
Submit: 1567  Solved: 664
[Submit][Status][Discuss]

Description

Input

第一行一个正整数，表示数据组数据 ，接下来T行
每行一个正整数N

Output

2*T行
第2*i-1行表示第i个数据中问题一的解，

第2*i行表示第i个数据中问题二的解，

Sample Input

1
1

Sample Output

1
2

HINT



x=1与x=2都是原方程的根，注意第一个问题的解

不要mod 10^9+7


1<=N<=10^18

1<=T<=1000
